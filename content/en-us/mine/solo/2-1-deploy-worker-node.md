## Optional: Pre-Sync

> *Optional:* You cannot start mining before your miner has been verified (fully synced). But you can execute the command below to pre-sync your Khala node.

 ```bash
 sudo phala presync
 ```

## Start Mining

Let's start mining!

```bash
sudo phala start
```
